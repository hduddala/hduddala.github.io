Replic-AI / BrandPilot

AI Social Media Agent for Startups

Replic-AI (BrandPilot) is an AI-powered social media agent that helps early-stage startups maintain an active and authentic online presence without a full-time marketer. The system monitors X (Twitter) and Reddit for brand mentions and keywords, automatically generating on-brand replies and daily posts. Built in a weekend at HackPrinceton 2025, it features a multi-brand dashboard with real-time activity logs and chat-native approval workflows that let founders approve, edit, or reject posts directly from iMessage.

Social Media Monitoring

Continuously monitors X and Reddit for brand mentions and relevant keywords.

AI Content Generation

Auto-generates on-brand replies and daily posts using xAI Grok for authentic engagement.

Multi-Brand Dashboard

Real-time activity logs and management interface built with Next.js and Supabase.

Chat-Native Approvals

Founders can approve, edit, or reject posts directly from iMessage via Photon/WhatsApp APIs.

FastAPI Backend

Robust Python backend with Redis caching for efficient data processing and API responses.

Composio Integration

Seamless workflow automation connecting social platforms and messaging services.
